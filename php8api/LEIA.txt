Status: Funcionando.

Foram utilizados: XAMPP 8.2.0 ; PHP 8.2.0; MYSQL 8.0.32.
Para testes: Postman.

Título: API em PHP para CRUD de usuários.

Descrição: Esse projeto foi feito com o objetivo de fazer um CRUD de usuários. 

Start: Antes de iniciar, vá ao arquivo "db_connection" dentro da pasta "config" e defina suas credenciais do banco de dados.

        private $host = "localhost";       ----- no lugar de "locahost" coloque o host de sua preferência.
        private $database_name = "apiphp"; ----- no lugar de "apiphp" coloque o seu banco de dados.
        private $username = "root";        ----- no lugar de "root" coloque o seu usuário.
        private $password = "884316";      ----- no lugar de "" coloque a senha do seu banco de dados

OBS: As sugestões de query para melhor funcionamento da api estão dentro do arquivo "QUERY API".
